🍽️ Restaurant Management System

A full-stack Restaurant Management System designed to manage orders, reservations, billing, inventory, and user authentication efficiently.

🚀 Features
👥 User Management (Admin, Staff, Customer)
🍔 Menu & Category Management
🧾 Order & Billing System
🪑 Table & Reservation Management
📦 Inventory Tracking
🔐 Authentication (Login, Password Reset, Remember Me)
📊 Order Status Tracking
